# TAKENOS: API WITH NESTJS

## Description

API NEST to consult about cryptocurrencies.

## Requirements
Node v18.17.0 or more.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
